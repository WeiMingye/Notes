# 在文件间跳转
## 相关搜索配置
命令|用途
---|---
set path+={path}|添加路径｛path｝
set suffixesadd+=.bat|添加扩展名.bat

## 跳转列表
- 凡是大范围的动作命令都算是跳转
- 文件间的跳转也会记录在跳转列表中
- 每个窗口都有独立的跳转列表

命令|用途
---|---
:jumps|查看跳转列表
<C-o>|后退到上一个跳转
<C-i>|前进到下一个跳转

## 改变列表
- 每次修改，vim都会记录当时的光标位置
- u<C-r>作用域g;相似
- 位置标记是修改列表的一个补充

命令|用途
---|---
g;|回退到前一个修改的地方
g,|前进到后一个修改的地方
gi|回到上次退出插入模式的位置继续修改


## 跳转到光标下的的文件
- gf 跳转到光标下的文件
- 可以使用suffixesadd和path添加扩展名以及路径
