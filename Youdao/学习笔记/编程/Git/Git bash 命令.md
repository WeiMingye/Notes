# Git Bash
## 快捷键
- <C-h>删除前一个字符
- <C-w>删除前一个单词
- <C-u>删除至行首
- <C-o>打开命令
- <C-r>进入搜索命令模式
- <C-a>返回正常模式
- <C-n>啥看下一条命令
- <C-p>查看上一条命令
- <C-t>当前字符与上一字符交换

## 常用命令
命令 | 用途
--- | ---
pwd（Print Working Directory） | 显示当前目录
ls（List） | 显示当前目录的文件（dos的dir可以）
mkdir（Make Directory）| 创建目录
touch | 创建文件
rmdir | 删除目录
rm | 删除文件
mv | 移动文件并且重命名
cp | 复制文件
wget | 下载
cd | 打开目录
cat | 查看文件全部内容
head | 查看文件开头n行
tail | 查看文件结尾n行
history | 查看历史
